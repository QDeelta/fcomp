# FCOMP

A statically-typed non-strict pure-functional programming language with a S-expression like syntax.


### Basics

- [x] Parser
- [ ] Type checker
- [ ] Intepreter
- [ ] Compile to some abstract machine code (e.g. G-machine)
- [ ] Garbage collector
- [ ] Compile to C
- [ ] Compile to LLVM IR


### Features

- [ ] Let bindings
- [ ] Polymorphism
- [ ] Type inference
- [ ] Monadic IO
- [ ] Bootstrapping
