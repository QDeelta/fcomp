# FCOMP

A statically-typed non-strict pure-functional programming language with an S-expression like syntax.

### Basics

- [x] Parser
- [x] Type checker
- [x] Compile to some abstract machine code (e.g. G-machine)
- [x] Interpreter
- [x] Compile to C
- [ ] Garbage collector

### Features

- [x] Integer Primitives
- [ ] Let bindings
- [ ] Pattern guards
- [ ] Polymorphism
- [ ] Type inference
- [ ] Monadic IO
- [ ] Bootstrapping
